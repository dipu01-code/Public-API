# 🌐 Public API Playground

A simple and interactive web application that demonstrates how to work with multiple public APIs using HTML, CSS, and JavaScript.

## 📌 Project Objective

This project combines multiple public APIs into a single user-friendly interface.  
It demonstrates API integration, asynchronous JavaScript, and responsive UI design.

---

## 🧩 Features

### 🐶 Dog Finder
- Fetches a random dog image
- Extracts and displays breed name from image URL
- Copy image URL to clipboard

### 😂 Joke Generator
- Displays a random joke
- Shows setup and punchline
- "Next Joke" button for continuous jokes

### 👤 Random User Profile
- Fetches random user data
- Displays:
  - Full name
  - Profile picture
  - Email
  - Country
  - Age/Phone (extra feature)

### 📄 JSONPlaceholder Viewer
- Fetches sample posts
- Displays post titles

---

## 🛠️ Tech Stack

- HTML5
- CSS3
- JavaScript (ES6)
- Fetch API
- Async/Await

---

## 🌍 APIs Used

- Dog API → https://dog.ceo/
- Joke API → https://official-joke-api.appspot.com/
- Random User API → https://randomuser.me/
- JSONPlaceholder → https://jsonplaceholder.typicode.com/

## ⚙️ Functional Requirements

- API calls using `fetch()`
- Asynchronous handling using `async/await`
- Loading state handling
- Error handling
- Button-triggered API calls
